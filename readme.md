@Descripción:

Este proyecto es una página web básica desarrollada como parte de un curso de Front-End. Tiene como temática principal ser un E-Commerce basado en la serie "Los Simpsons", tomando como referencia la famosa tienda de dicha serie: el "Kwik-E-Mart". La página está estructurada con HTML semántico y utiliza etiquetas para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript. Eventualmente, debería adquirir funciones con JavaScript para simular el comportamiento de una página de E-Commerce en la que se pueda comprar productos.

Tecnologías: HTML - CSS - JavaScript
